Introduction

The Internet is the largest communication system in the world. It is a collection of interconnected networks that enables communication and resource sharing among millions of devices.

Networks

  • Host Devices (Computers, Laptops, Phones)
  • Connecting Devices (Routers, Switches, Modems)
  • Transmission Media (Wired and Wireless)

Local Area Network (LAN)

  • Small geographical coverage
  • Privately owned network
  • High-speed communication
  • Uses switches for packet forwarding

Wide Area Network (WAN)

  • Large geographical coverage
  • Connects routers and switches
  • May span cities, countries and continents
  • Can be Point-to-Point or Switched WAN

Internetwork

An internetwork is formed when multiple networks are connected together using routers.

Switching

  • Circuit Switching
  • Packet Switching

The Internet primarily uses packet switching.

The Internet Structure

  • Backbone Networks
  • Provider Networks
  • Customer Networks
  • Internet Service Providers (ISPs)

Internet Access Technologies

  • Dial-Up
  • DSL
  • Cable Internet
  • Wireless Internet
  • Dedicated Connections

Hardware and Software

  • Computers
  • Routers
  • Switches
  • Protocols
  • Applications

Summary

The Internet is a global collection of interconnected networks built using LANs, WANs, routers, switches, transmission media, hardware, software, and communication protocols.
